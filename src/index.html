Markdown to AB-flavored BBCode

dev build
Markdown
BBCode

Image + Title text

![alt-text](link) becomes [img=alt-text]link[/img]

Alt-text portion can be left blank.

Urls

[memes](http://ksmz.moe) becomes [url=http://ksmz.moe]memes[/url]

"Bigger" h1 and h2

memes
===
V
[quote][size=7]memes[/size][/quote]

(and --- will produce size 6)

Regular h1 to h5

# Memes becomes [size=7]memes[/size]

No h6 because that's the default size on AB (size 2).

Code

Backticks (```), Tilde(~~~) and Inline (`) works. Inline is sorta useless because AB sets [code] to always wrap.

Stylization

Standard *italic*, **bold** and ***both*** work.

Quote

> to quote. Works on a single line.

Lists

Star, Minus and Plus lists output regular dotted lists [*].

Numbered lists will give numbered lists (duh) [#]
